trokko
====

Embedded operating system based on KOZOS & uITRON

## Target board

* H8/3069F

## Requirement

* [binutils-2.19.1](http://www.gnu.org/software/binutils/)
* [gcc-3.4.6](https://gcc.gnu.org)
* [kz_h8write](https://osdn.jp/projects/kz-h8write/)
